📊 Retail Sales & Revenue Dashboard
📌 Project Overview

This project focuses on analyzing retail sales data and building an interactive dashboard to uncover business insights related to revenue, profitability, customer segments, product categories, regions, and shipping modes.

🛠️ Tech Stack
Python
Pandas
Matplotlib
Google Colab
Google Looker Studio
🔄 Project Workflow
Data Collection
Data Cleaning using Pandas
Data Transformation & Feature Preparation
Exploratory Data Analysis (EDA)
Dashboard Development in Google Looker Studio
Business Insight Generation

📈 Dashboard Highlights
Total Sales KPI
Total Profit KPI
Total Orders KPI
Total Quantity KPI
Profit Margin KPI
Sales & Profit Trend Analysis
Segment-wise Sales Distribution
Category-wise Sales Analysis
Region-wise Profit Analysis
Product Performance Analysis
Discount vs Profit Analysis
Ship Mode Performance Analysis
Interactive Filters

💡 Key Insights
Technology category generated the highest revenue.
Consumer segment contributed the largest share of sales.
Higher discounts reduced profitability.
Standard Class was the most frequently used shipping mode.
Top products significantly influenced overall revenue.
